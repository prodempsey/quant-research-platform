# quant-research-platform

Phase 1 of `quant-research-platform` is a personal ETF tactical research and internal paper-portfolio platform. It is not a live trading system. Phase 1 excludes live broker integration, individual stocks, fundamentals, ETF holdings, news / events / earnings, options data, and autonomous research agents.

Project documentation lives under [`docs/`](docs/), with the Strategy Decision Record at `docs/strategy_decision_record.md`, the Engineering Workflow at `docs/engineering_workflow.md`, and the Phase 1 Engineering Specification under `docs/engineering_spec/`.

Implementation is governed by the Governor-Gated GitHub PR Agent Loop (`docs/runbooks/governor_gated_github_pr_agent_loop.md`). This repository is currently in scaffold state; no business behavior is implemented yet.
